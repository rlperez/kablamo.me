## Development

### Install dependencies
This project uses [bun](https://bun.sh/)

```
bun install
```

### Working locally

Starts watch tasks to compile when changes detected

```
bun run dev:11ty
```

### Creating a production build

Minify JS, CSS and HTML.

```
bun run build
```
